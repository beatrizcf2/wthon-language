# Wthon
This language is a global language. This project will implement thre different idioms: English, German and Portuguese.

# Example

```py
--english;
def HelloWorld(int n){
    int x;
    x = 0;
    while (x < n){
        print("Hello World: ");
        print(x);
        x = x + 1;
    };
};
HelloWorld(5);


--portugues;
def OlaMundo(int n){
    int x;
    x = 0;
    enquanto (x < n){
        imprime("Ola Mundo: ");
        imprime(x);
        x = x + 1;
    };
};
OlaMundo(5);


--deutsch;
def HalloWelt(int n){
    int x;
    x = 0;
    solange (x < n){
        drucke("Hallo Welt: ");
        drucke(x);
        x = x + 1;
    };
};
HalloWelt(5);
```
